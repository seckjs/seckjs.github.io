## 新手提示

在这个插件里，真正决定体验顺不顺的主要是四件事：

1. 扩展是否已经正确加载到浏览器。
2. AI 设置是否完整，尤其是 `endpoint`、`API Key`、`model`。
3. 是否已经在超星相关页面中，让插件拿到了课程和作业上下文。
4. 一键批阅前是否已经打开自动填写、自动提交，以及正确的提交方式。
